######################
Command Line Interface
######################

This page list all of the available command line flags for the :command:`euporie` command.

The :command:`euporie` command is included in the ``euporie`` metapackage, and provides a launcher for the various euporie apps (such as :command:`euporie-console`, :command:`euporie-notebook`, :command:`euporie-preview`, and :command:`euporie-hub`). It allows you to start any of the euporie applications through a single unified entry point.
